* As a user I want to be able to search through the database based on my input query:
  When input is valid, correct corresponding result should be displayed.
  When input is invalid, no result should be displayed.
  When wrong information been put into the wrong section, no result should be displayed.
  When BACKSPACE button is pressed, the input query should be deleted.
  When result being displayed, user should be able to view all the necessary statistics.
  When information is not present, no result should be shown.
  When information is present, and it has not been called, it should not be shown.
  When information is present, and it has been called correctly, it should be displaying.
